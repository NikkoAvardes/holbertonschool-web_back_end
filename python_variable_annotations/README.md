# Annotations de variables en Python

Ce projet présente des exemples simples d’utilisation des annotations de type pour les variables en Python. Chaque fichier illustre une tâche distincte liée aux types, fonctions et variables.

## Contenu

- Exemples d’annotations de type pour variables, fonctions et listes
- Exercices courts pour pratiquer
- Fichiers de test pour vérifier les solutions

## Utilisation

Lancez les fichiers `./*-main.py` pour tester les solutions.
