ES6 Basics

Ce projet couvre les fondamentaux d'ES6 (ECMAScript 2015) :

Constantes et variables : const vs let vs var, block scope
Fonctions fléchées : syntaxe () => et contexte this
Paramètres par défaut : valeurs par défaut dans les fonctions
Rest/Spread : ...args pour les paramètres et tableaux
Template literals : interpolation de chaînes avec backticks
Propriétés abrégées : syntaxe courte pour les objets
Computed property names : noms de propriétés dynamiques
Method properties : méthodes dans les objets littéraux
Itérateurs : for...of loops
Structures de données : manipulation d'objets et tableaux
Chaque exercice (0-12) introduit progressivement ces concepts ES6 modernes.
